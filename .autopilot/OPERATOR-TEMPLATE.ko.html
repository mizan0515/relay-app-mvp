🧘 이건 관리자 일이 아니에요

아래 항목들은 오토파일럿이 알아서 처리합니다. 눈에 띄어도 무시하세요.

    📚 최근 작업 흐름

    • —
    📖 용어와 파일 설명 (클릭해서 펼치기)
    iter / iteration (반복 회차)
    오토파일럿이 "한 번 일한 단위"입니다. 숫자가 늘어나면 그만큼 일했다는 뜻.
    /loop .autopilot/PROMPT.md
    클로드 코드 채팅에 입력하면 오토파일럿이 시작/재시작됩니다. 멈췄을 때만 필요해요.
    HALT
    정지 신호 파일. 있으면 루프가 멈추고, 지우면 다시 돕니다. .\.autopilot\관리자.ps1 재개로 지울 수 있어요.
    ScheduleWakeup / LAST_RESCHEDULE
    다음 깨어남을 예약하는 내부 장치. 고장나면 이 화면이 🚨로 알려주고 고치는 법도 알려줍니다.
    PR (Pull Request)
    깃허브에 코드 변경을 올리는 상자. 오토파일럿이 만들고 자동으로 머지까지 합니다. 관리자는 "리뷰해달라"고 뜬 PR만 손대면 됩니다.
    STATE.md / HISTORY.md / BACKLOG.md
    각각 "현재 상태", "최근 한 일", "해야 할 일 목록". 관리자는 열어볼 필요 없어요. 이 화면이 대신 읽어줍니다.
    ⌨️ 명령 레퍼런스 (필요할 때만)
    시작/재시작
    클로드 코드 채팅에 /loop .autopilot/PROMPT.md 입력.
    정지
    PowerShell에서 .\.autopilot\관리자.ps1 정지.
    재개
    PowerShell에서 .\.autopilot\관리자.ps1 재개.
    대시보드 새로고침
    관리자 대시보드 열기.cmd 더블클릭.
    이 화면은 자동 갱신되지 않습니다. 새로 보려면 관리자 대시보드 열기.cmd 를 다시 더블클릭하거나, 오토파일럿이 다음 반복을 마치면 자동 갱신됩니다.
    모든 커밋·PR·머지·브랜치 정리는 오토파일럿이 합니다. 관리자가 손댈 일은 이 화면 맨 위 "지금 해주실 일"에만 나옵니다.
